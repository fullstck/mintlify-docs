<Update label="July 2025">
  ## New Property Location Features

  ### GPS Precision with geoAccuracy
  
  The new `geoAccuracy` field indicates GPS coordinate precision. Value 1 = precise GPS coordinates for optimal georeferencing.

  ### withLocation Filter
  
  Instantly filter properties with GPS coordinates using the `withLocation` parameter. Optimize your geographic searches.

  ### Polygon Search
  
  Define custom search areas with polygon search. Perfect for targeting specific neighborhoods and improving result relevance.
</Update>
